# 방송국 뷰어 서비스 (Broadcast Viewer Service)

## 프로젝트 개요

아프리카TV, 숲TV, 팬더TV, 팝콘TV 등 주요 방송 플랫폼의 방송을 시청하는 뷰어 서비스 웹사이트입니다. 

이 프로젝트는 GitHub Pages에 호스팅되어 있으며, SEO 최적화를 통해 "아프리카tv뷰어", "숲tv뷰어", "팬더tv뷰어", "팝콘tv뷰어" 키워드로 Google 검색에 잘 노출되도록 설계되었습니다.

## 🌐 라이브 웹사이트

**URL**: https://kimafreeca.github.io/afreeca/

## 🎯 주요 기능

### 1. 다중 플랫폼 지원
- **아프리카TV 뷰어**: 방송 시청, 구독자 증가
- **숲TV 뷰어**: 방송 시청 뷰어 서비스
- **팬더TV 뷰어**: 방송 시청 서비스
- **팝콘TV 뷰어**: 방송 시청 뷰어

### 2. 뷰어 서비스 기능
- 24시간 중단 없는 운영
- 방송 시청
- 채팅 참여
- 시청 시간 증가
- 예약 기능

### 3. 사용자 인터페이스
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 직관적인 네비게이션
- 인터랙티브 요소
- 실시간 서비스 상태 표시

### 4. 문의하기 기능
- **카카오톡 문의**: 실시간 상담 가능
- **텔레그램 문의**: 빠른 응답 보장
- 문의하기 버튼 클릭 시 모달로 연락처 선택 가능
- 365일 24시간 연중무휴 지원
